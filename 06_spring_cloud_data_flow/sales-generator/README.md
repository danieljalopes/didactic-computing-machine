# Sales Generator
This a simple data generator application aimed to generate fake data to be consumed for any application/service.
Its build with [Spring WebFlux](https://docs.spring.io/spring-framework/reference/web/webflux.html). And is intended to run in a Docker container.

# Requirements
[Docker](https://www.docker.com/)


# How to build
Open terminal on this folder
run
````bash
./gradlew 
````
